Portafolio

Proyectos que hemos construido para nuestros clientes.

PROYECTO 01

Sistema Centralizado de QA para Radiodiagnóstico y Medicina Nuclear

Plataforma centralizada de QA para todas las modalidades de radiodiagnóstico y medicina nuclear.

    PROYECTO 02

    Análisis Continuo de Niveles de Referencia de Dosis (DRL)

    Herramienta conectada al PACS institucional para monitoreo continuo de dosis y optimización frente a estándares DRL internacionales.

      PROYECTO 03

      Sistema de Gestión de Registros y Firmas para Instituciones Radiológicas

      Sistema de registros y firma digital para el control de actividades diarias del personal radiológico, con datos confirmados inmutables.

        PROYECTO 04

        Plataforma Integral de Gestión de Protección Radiológica

        Plataforma integral para inventario de equipos radiológicos, alertas automáticas de QA y seguimiento dosimétrico del personal expuesto.

          PROYECTO 05

          Calculadora de Dosis Efectiva para Pacientes Pediátricos

          Calculadora especializada de conversión de DLP a dosis efectiva en pacientes pediátricos, con interpolación entre tablas internacionales.
